# Components list

The `ComponentsList` showcases all the components that can be used in the form designer. The list is
populated with the component groups and component definitions that are defined in the backend
application, and a collection of custom-defined preset component definitions.

The backend defined components and custom-defined preset components are displayed using the
`RegularComponentsGroup` and `PresetComponentsGroup` components. These wrapper components are
responsible for turning their respective component definitions into a standardized component
structure that can be rendered by the `GenericComponentsGroup` component. The
`GenericComponentsGroup` contains the actual component rendering logic for filtering, and (in the
future) the drag-and-drop functionality.

The UI and functionality of the `ComponentsList` component is based on the react-formio FormBuilder
component.

## Component groups

The `ComponentsList` component groups the components by the following groups:

- **Regular** – the basic form components (e.g., text fields, checkboxes)
- **Special** – more specific and/or complex form components (e.g., BSN, repeating groups, partners)
- **Layout** – structural components (e.g., columns, fieldsets)
- **Preset** – components that have predefined HTML autocomplete attributes.

The component groups, and the components within these groups, are managed by the backend
application.

## Search functionality

The `ComponentsList` component contains a search field which filters components by both their
technical name and display label. The actual filtering logic is implemented in the
`GenericComponentsGroup` component.

## (Future) usage

The `ComponentsList` component should be used in conjunction with the `FormDesigner` component.

The `GenericComponentsGroup` will provide the source of draggable components that can be used in the
form designer to construct a form definition. These draggable components should be dropped in the
upcoming `FormDesigner` component, which will be responsible for rendering the form definition.
